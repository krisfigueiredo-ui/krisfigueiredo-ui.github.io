# Kristofor Figueiredo - Portfolio

Personal portfolio for Kristofor Figueiredo, focused on business analytics, technology sourcing, supply chain, sports analytics, and applied machine learning.

**Live site:** https://krisfigueiredo-ui.github.io/

## Local preview

```bash
python3 -m http.server 8000
```

Open `http://127.0.0.1:8000/`.

## Content and privacy

The site is based on the owner-provided resume and links only to verified public repositories. It intentionally excludes confidential client work, unpublished source data, private contact details, and proprietary deliverables. Team projects name the classmates credited in the source project materials.

## Stack

- Semantic HTML
- Responsive CSS
- No framework, tracking, external font, or build dependency
- GitHub Pages deployment through Actions
